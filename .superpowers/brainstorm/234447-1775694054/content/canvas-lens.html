Nested groups as first-class nodes
Groups render their subgroups inline as pills inside the node — not as separate graph nodes, which would create visual chaos. Click a subgroup pill to jump to it. Members of a subgroup automatically inherit parent group grants. The Reasoner traces the full chain: Weekend Crew → NOC Team → Night Ops grant.
📅
Schedules as connectable nodes
Schedules live on the canvas as first-class nodes. You draw an edge from a Schedule to a Grant to attach it — the grant then only activates during those windows. Holidays are shown as small exclusion tags on the schedule node. Hovering shows the full calendar.
📋
Click-to-edit without leaving the graph
Selecting any node opens the detail panel — no page navigation. The panel shows live membership rules, attached grants, and schedule in a readable format. Edit group opens a slide-over form. Query in Oracle and Trace in Reasoner keep the canvas context in view.
