Forward + reverse queries
Two modes: "Who can access [door] at [time]?" evaluates all users against a door. "What can [person] access right now?" evaluates one user against all doors. Both run the real ABAC engine — not pre-computed.
🏖️
Holiday simulation built in
A toggle lets you simulate any query as if today were a public holiday. Instantly shows who loses access under holiday rules — critical for understanding gaps in your holiday coverage before Christmas Eve.
→
Every result links to Reasoner
Each row shows a one-line reason summary inline. Click "Trace" on any result and the Reasoner opens pre-loaded with that exact person + door + time — no re-entering context.
