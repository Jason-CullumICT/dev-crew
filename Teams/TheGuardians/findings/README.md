# Guardian Findings

Security and compliance reports generated by TheGuardians.

## File Convention

| File | Description |
|------|-------------|
| `security-report-{date}-{grade}.html` | Full HTML security and compliance report |
| `security-backlog-{date}.json` | Machine-readable Critical/High findings for TheFixer |

## Severity Mapping to TheInspector Scale

| TheGuardians | TheInspector | TheFixer Priority |
|---|---|---|
| Critical | P1 | Block deployment |
| High | P2 | This sprint |
| Medium | P3 | Next sprint |
| Low | P4 | Backlog |

When both teams run against the same codebase, deduplicate by finding ID before feeding to TheFixer.
