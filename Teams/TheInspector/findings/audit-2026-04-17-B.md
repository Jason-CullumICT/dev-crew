#### DEP-001: Handlebars.js Critical Code Injection (Multiple CVEs)
- **Severity:** P1 (CRITICAL)
- **Category:** CVE — JavaScript Code Injection
- **Package:** `handlebars` (transitive, via build/template tools)
- **Current Range:** `4.0.0 – 4.7.8`
- **Required Fix:** Upgrade to `>=4.7.9`
- **CVE IDs:**
  - GHSA-2qvq-rjwj-gvw9: Prototype Pollution → XSS via partial template injection
  - GHSA-2w6w-674q-4c4q: AST Type Confusion → arbitrary code execution (CVSS 9.8)
  - GHSA-3mfm-83xf-c92r: AST Type Confusion → code injection (CVSS 8.1)
  - GHSA-442j-39wm-28r2: Property access validation bypass (CVSS 3.7)
  - GHSA-7rx3-28cr-v5wh: Prototype method access control gap (CVSS 4.8)
  - GHSA-xhpv-hc6g-r9c6: AST Type Confusion via dynamic partial (CVSS 8.1)
  - GHSA-xjpj-3mr7-gcpf: CLI precompiler JS injection (CVSS 8.3)
  - GHSA-9cx6-37pm-9jff: Malformed decorator DoS (CVSS 7.5)
- **Impact:** Allows attacker-controlled template content to execute arbitrary JavaScript
- **Fix:** `npm install handlebars@latest`
- **Cross-ref:** [ESCALATE → TheGuardians] — Requires code review of template handling
